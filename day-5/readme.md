# Day 5 - Loop

## Tujuan
    Memahami konsep loop, array

## Materi
    while loop
    do..while loop
    for of loop
    for in loop
    array
    array multidimensi
    array of object
## Contoh

## Insights
    while loop adalah melakukan loop ketika kondisi terpenuhi
    do....while loop adalah melakukan setidaknya sekali, meskipun kondisi salah
    for of loop adalah mengambil data dalam array/objek
    for in loop adalah mengambil key/properti dalam array/objek
    array adalah tempat menyimpan data/elemen dalam sebuah variabel
    array multidimensi menyimpan data/elemen dalam bentuk matriks
    array of object menyimpan objek sebagai elemen
