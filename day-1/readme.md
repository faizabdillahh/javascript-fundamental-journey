# Day 1 - Variabel & Tipe Data

## Tujuan
    Memahami variabel
    Memahami tipe data dasar

## Materi
    variabel
    String, Number, Boolean, Operasi dasar
    Type Coercion

## Contoh
    const nama = "faiz";
    let umur = 17;
    console.log(`Hallo nama saya ${nama}, umur saya ${umur} tahun.`);

## Insight
    variabel adalah tempat untuk menyimpan data
    tipe data adalah jenis data yang digunakan
    didalam javascript ada proses bernama type coercion yaitu proses otomatis mengganti data string menjadi int pada proses matematika
    pengggunaan === adalah yang terbaik daripada == untuk menghindari error yang disebabkan type coercion
