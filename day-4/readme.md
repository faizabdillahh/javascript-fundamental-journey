# Day 4 - For Loop

## Tujuan
    Memahami konsep for loop

## Materi
    for loop

## Contoh
for (let i = 0; i > 10; i++) {
    console.log(`angka ${i}`);
}

## Insights
    loop adalah cara untuk mengulang ngulang suatu proses dengan code
    for loop bisa digunakan untuk validasi password
    struktur for loop itu for(initialization; condition; increment) {
        //code yang mau diulang
    }
    intitialization = membuat variabel penghitung, hanya jalan satu kali diawal
    condition = kondisi yang harus dipenuhi untuk diulang ulang
    increment = menambah nilai proses yang diulang ulang atau bisa dibilang mesin penggeraknya
