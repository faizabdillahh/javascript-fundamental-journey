# Day 2 - If Else

## Tujuan
    Memahami konsep if else

## Materi
    IF
    ELSE
    ELSE IF

## Contoh
let umur = 17;
if (umur >= 18) {
    console.log("anda sudah dewasa);
} else (umur < 18) {
    console.log("anda belum dewasa);
}

## Insight
    if else adalah baris kode yang melakukan sesuatu bila memenuhi kondisi yang ditentukan
